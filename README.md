# PCB-designing-
A collection of PCB designs made during my hardware learning journey — Power Supplies, Data Logger &amp; more. Schematics, Gerbers &amp; 3D views included.
This repository contains all the PCB projects I designed 
while learning PCB design from scratch using KiCad.

Each folder contains:
- Schematic (PDF)
- 3D View (PNG)
- Gerber Files

Projects included:
1. Breadboard Power Supply
2. Elektor Tiny Power Supply
3. Transformerless Power Supply
4. ATmega328P Data Logger

Still learning — feedback & suggestions are always welcome!
