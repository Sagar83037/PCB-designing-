# Transformerless Power Supply

A compact **220V AC to regulated 5V DC transformerless power supply** using a capacitive dropper, bridge rectifier, and **LM7805 voltage regulator**. Suitable for **low-power electronics and embedded circuits**.

**Warning:** This circuit is directly connected to mains and is **not isolated**.

# Components

| Reference | Component | Description |
|-----------|-----------|-------------|
| C1 | 2.2µF Capacitor | Capacitive dropper |
| R1 | 1MΩ Resistor | Discharge resistor |
| D1–D4 | 1N4007 | Bridge rectifier diodes |
| C2 | 0.1µF Capacitor | Filtering |
| C3 | 100µF Capacitor | Smoothing capacitor |
| D5 | Zener Diode | Voltage protection |
| D6 | Diode | Reverse protection |
| R2, R3 | 20kΩ | Bias resistors |
| R4 | 2.2kΩ | LED current limit |
| Q1 | LM7805 | 5V voltage regulator |
| LED1 | LED | Power indicator |
| J1 | Terminal | AC input |
| J2 | Terminal | 5V DC output |

# Output Voltage

Regulated using **LM7805 → 5V DC output**
